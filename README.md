Technologies used

  HTML, 
  CSS, 
  Javascript, 
  React js, 
  NodeJs, 
  axios, expressJs,cors (packages installed),

Commands,
to run front end

  >cd client
  
  >npm start

to run back end

  >cd server 
  
  >node index.js

to install packages

  >npm install <package name>
